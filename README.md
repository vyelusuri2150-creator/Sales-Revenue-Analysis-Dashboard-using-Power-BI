# Sales-Revenue-Analysis-Dashboard-using-Power-BI
Project Overview

This project analyzes sales and revenue data using Power BI. The dashboard provides insights into sales performance, revenue trends, and top-performing products.

Tools Used
Power BI
Excel/CSV Dataset
Dashboard Features
KPI Cards for Total Sales, Profit, and Orders
Revenue Trend Analysis
Top Performing Products
Interactive Filters and Slicers
Key Insights
Identified top revenue-generating products.
Analyzed monthly sales trends.
Compared sales performance across categories.
